## Installation

API Used : [News API](https://newsapi.org/). 

Dependencies:
* [Flutter HTTP](https://pub.dev/packages/http)
* [Flutter SpinKit](https://pub.dev/packages/flutter_spinkit)
* [Flutter Webview](https://pub.dev/packages/flutter_webview_plugin)


Clone the repo

```bash
flutter pub get
flutter run
```
